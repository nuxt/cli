## Usage

### Install globally

```bash
npm i -g nuxi-ng
```

Run your projects with new CLI:

```bash
nuxi dev
```

### Update for project

Install new CLI dependency:

```bash
# pnpm
pnpm add -D nuxi-ng

# yarn
yarn add -D nuxi-ng

# npm
npm i -D nuxi-ng
```

Change scripts:

```json
{
  "scripts": {
    "dev": "nuxi-ng dev",
    "start": "nuxi-ng start",
    "build": "nuxi-ng build",
    "generate": "nuxi-ng generate"
  }
}
```
